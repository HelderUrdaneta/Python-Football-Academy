¡Bienvenido a Python Soccer Code!

Aprende a programar en Python resolviendo desafíos de fútbol.

    Guía Completa
    El Objetivo

    Tu meta es escribir código Python en el editor de la izquierda para guiar a tu jugador a completar la misión de cada nivel en el campo de la derecha.

    Controles de Ejecución
    • Ejecutar: Corre tu código de principio a fin con una animación visual.
    • Siguiente Paso: Ejecuta solo una línea de tu código. Ideal para depurar y ver el flujo exacto.
    • Detener: Para cualquier ejecución en curso.
    • Reiniciar Nivel: Devuelve al jugador y al balón a sus posiciones iniciales sin cambiar tu código.
    Comandos de Programación

    Estos son los bloques de construcción de tu programa.

    Las acciones hacen que tu jugador haga algo en el campo.

    avanzar() - Mueve al jugador una casilla hacia adelante.
    girar_izquierda() - Gira al jugador 90° a la izquierda.
    recoger_balon() - Toma el balón si está en la misma casilla.
    soltar_balon() - Deja el balón en la casilla actual.

    Consejo:

    # Puedes crear nuevas funciones combinando otras.
def girar_derecha():
    girar_izquierda()
    girar_izquierda()
    girar_izquierda()

    Las condiciones te permiten hacer preguntas para tomar decisiones con if o while.

    frente_despejado() - Devuelve `True` (verdadero) si no hay un obstáculo en frente.
    tiene_balon() - Devuelve `True` si tu jugador ya tiene el balón.

    Ejemplo de uso:

    # Solo avanza si el camino está libre.
if frente_despejado():
    avanzar()
else:
    # Si no, gira.
    girar_izquierda()

    Usa bucles para repetir acciones y condicionales para crear programas inteligentes.

    Bucle for (repetir N veces): 
    # Esta función hace que el jugador avance 4 veces.
def avanzar_cuatro_casillas():
    for i in range(4):
        avanzar()
    Bucle while (repetir mientras algo sea verdad): 
    # Avanza sin parar hasta que encuentre un obstáculo.
while frente_despejado():
    avanzar()
