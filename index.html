Ayuda
Funciones Disponibles:
  • avanzar() - Mueve al jugador una casilla en la dirección actual
  • girar_izquierda() - Gira al jugador 90° a la izquierda
  • recoger_balon() - Recoge el balón si está en la misma casilla
  • soltar_balon() - Deja el balón en la casilla actual
Consejos:
  • Puedes crear tu propia función girar_derecha() usando tres girar_izquierda()
  • Utiliza bucles para repetir acciones
  • Usa condicionales para tomar decisiones
Ejemplos:

def avanzar_tres():
    avanzar()
    avanzar()
    avanzar()
